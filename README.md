## Fruit Inspection

This is a project work for Computer Vision and Image Processing course at the University of Bologna.
The aim of this project was to solve 3 main tasks.
The file [fruit-inspection.pdf](./fruit-inspection.pdf) describes the three tasks in detail.
The folder [fruit-inspection-images](./fruit-inspection-images/) contains the images used throughout the project.
Finally, the notebook files contains the solutions for each tasks.

### First task
The first task was about fruit segmentation and defect detection. 

### Second task
In the second task the goal was about russet detection.

### Final task
The final task focused on kiwi inspection.
